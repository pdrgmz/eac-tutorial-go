Ambiente

Editor de texto

Puedes usar el editor de texto que gustes, los archivos que creamos en él serán llamados fuentes, donde estara el codigo del programa, para Go deben estar llamados con la extensión ".go"

Compilador de Go

El código escrito en el fuente es para la lectura de un humano, es decir, que necesita ser compilado y transformado en lenguaje de máquina para que el computador pueda ejecutar las instrucciones dadas en el archivo. El compilador de Go permite esta transformación.

El Compilador puede ser instalado en FreeBSD, Linux, Mac OS X y Windows, tanto en arquitecturas de 32-bits y 64-bits.

Para instalarlo en nuestro computador debemos ir a la página de descarga oficial y seguir las instrucciones para el sistema operativo.

Tutorial

Este tutorial presenta una consola de comando, que interpreta el lenguaje de programación Go. Para usarla se debe escribir el código en la sección izquierda de la consola y al hacer click en el botón Execute se mostrar la salida del código escrito.

Estructura

  package main

  import (
  "fmt"
  "math"
  )
  /* Mi primer programa en Go */
  func main() {
  fmt.Println("Feliz día", math.Pi)
}
Un programa costa de las siguientes partes:
  • Declaración de paquete 
    package main

    Esta primera línea es obligatoria y define el nombre del paquete, ya que Go se ejecuta en paquetes, el paquete llamado main en el punto inicial de cualquier programa de Go

  • Importación de paquetes 
    import (
  "fmt"
  "math"
)

    En esta sección se están importando "fmt" y "math", que permiten utilizar funciones de dichos paquetes.

  • Funciones 
    func main()

    Esta es la funcion principal donde el programa inicia.

  • Comentarios 
    /* Mi primer programa en Go */

    Esta línea es ignorada por el compilador y es como se agregan comentarios al codigo, tambien se puede usar // para comentarios de una sola línea.

  • Instrucciones y expresiones 
    fmt.Println("Feliz día", math.Pi)

    Esta es una función disponible en Go que permite mostrar en pantalla la cadena "Feliz día" y el valor calculado por math.Pi.

Separador de línea

Go agrega internamente los separadores de línea luego de cada instrucción por lo que no es necesario agregar ";" como en otros lenguajes como C y Java. Por ejemplo: 
fmt.Println("Hola Mundo")
            fmt.Println("Estoy en un mundo de Go")

Identificadores

Un identificador es un nombre utilizado para identificar una variable, función o cualquier otro elemento definido por el usuario. Estos pueden empezar con cualquier letra en mayuscula o minuscula, seguido por ninguna o más letras, guiones bajos o números.

Go no permite el uso de caracteres de puntuación como @, $ y % en los identificadores.

Go es sensible a mayusculas y minusculas

Ejemplos:
mahesh      kumar   abc   move_name   a_123
myname50   _temp    j      a23b9      retVal

Variables

La sentencia 

var
declara una lista de variables; como en la lista de argumentos de las funciones, el tipo se indica al final. 

var x, y, z int
var c, python, java bool

La declaración de variables permite inicializaciones, una por variable. Si se inicializa una variable, el tipo puede omitirse; la variable adoptará el tipo del valor con el que ha sido inicializada. 

var x, y, z int = 1, 2, 3
var c, python, java = true, false, "¡no!"

Dentro de una función, puede utilizarse la sentencia de asignación := en lugar de la declaración var. (Fuera de una función, todas las declaraciones de variables comienzan con la palabra clave var y el operando := no está disponible.) 

c, python, java := true, false, "¡no!"

Constantes

Las constantes se declaran como las variables, pero con la palabra reservada const. Las constantes pueden ser cadenas, booleanas, o numéricas. 

const Pi = 3.14

Estructuras de control

Toma de decisión

La instrucción if es similar a la sentencia en C o Java, salvo que los paréntesis ( ) desaparecen (ni siquiera son opcionales) y las llaves { } son obligatorias. 
package main

      import (
      "fmt"
      "math"
      )

      func sqrt(x float64) string {
      if x < 0 {
      return sqrt(-x) + "i"
    }
    return fmt.Sprint(math.Sqrt(x))
  }

  func main() {
  fmt.Println(sqrt(2), sqrt(-4))
}
La sentencia if puede empezar con una instrucción de inicialización que se ejecutará antes de evaluar la condición. Las variables declaradas por la instrucción de inicialización son únicamente visibles en el ámbito del if. 
if v := math.Pow(x, n); v < lim {
  return v
}
Las variables declaradas dentro de la instrucción de inicialización de un if son también visibles dentro de los bloques else. 
if v := math.Pow(x, n); v < lim {
    return v
  } else {
  fmt.Printf("%g >= %g\n", v, lim)
}
La sentencia switch es también similar a C o Java, el cuerpo de un caso sale automáticamente de la cláusula switch, a menos que termine con una sentencia fallthrough, que provocaría que siguiera ejecutando el siguiente caso contemplado. 
switch os := runtime.GOOS; os {
  case "darwin":
  fmt.Println("OS X.")
  case "linux":
  fmt.Println("Linux.")
  default:
  // freebsd, openbsd,
  // plan9, windows...
  fmt.Printf("%s.", os)
}

Ciclos

Go tiene sólo un operando para definir los ciclos, los ciclos for. El ciclos for básico es muy parecido al que se utiliza en C o Java, salvo que los ( ) desaparecen (ni siquiera son opcionales) y las llaves { } son obligatorias. 
package main

    import "fmt"

    func main() {
    sum := 0
    for i := 0; i < 10; i++ {
    sum += i
  }
  fmt.Println(sum)
}
Como en C o Java, puedes dejar las instrucciones de inicialización e incremento vacías. 
for ;sum < 1000; {
      sum += sum
    }
Puedes eliminar los puntos y coma ;: Un ciclo while de C se transforma en un ciclo for en Go. 
for sum < 1000 {
  sum += sum
}
Si omites la condición del bucle, es un bucle infinito de manera que un bucle infinito se escribe de manera compacta. 
for {
    }

Funciones

Una función puede tener cero o más argumentos. 
package main

    import "fmt"

    func add(x int, y int) int {
    return x + y
  }

  func main() {
  fmt.Println(add(42, 13))
}
Cuando dos o más parámetros consecutivos de la función son del mismo tipo, puedes omitir el tipo de todos menos del último. Una función puede devolver varios resultados. Esta función devuelve dos cadenas. 
func swap(x, y string) (string, string) {
  return y, x
}
Las funciones tienen parámetros; en Go los resultados pueden ser nombrados y actuar como variables; se les denomina "variables de retorno" Si las variables de retorno tienen un nombre, una sentencia return sin argumentos devuelve el valor actual de dichas variables. 
func split(sum int) (x, y int) {
  x = sum * 4 / 9
  y = sum - x
  return
}

Datos estructurados

Estructuras

Una estructura (struct) es un registro de variables dentro de un mismo tipo.(Y una declaración type declara un nuevo tipo de datos.) 
package main

    import "fmt"

    type Vertex struct {
    X int
    Y int
  }

  func main() {
  fmt.Println(Vertex{1, 2})
}
Los campos de una estructura son accesibles mediante el operador . (punto).

Apuntadores

Go posee punteros, pero no tiene aritmética de punteros (como C). Los campos de las estructuras pueden accederse a través de un puntero a una estructura. La dirección del puntero es transparente al programador. 
package main

    import "fmt"

    type Vertex struct {
    X int
    Y int
  }

  func main() {
  p := Vertex{1, 2}
  q := &p
  q.X = 1e9
  fmt.Println(p)
}
